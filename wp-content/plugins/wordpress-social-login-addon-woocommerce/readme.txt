Adds PagSeguro gateway to the WooCommerce plugin

== Description ==

### Adds some functions to better advantage of the Woocommerce and Wordpress Social Login ###

Facebook Login
Use facebook data user to Woocommerce fields.
First Name -> Billing First name
Last Name -> Billing Last name
Birth -> Billing Birthdate
Gender -> Billing Sex (Just work for pt-br woocommerce translation)

-Other medias
Noting yet. 

Please notice that WooCommerce and Wordpress Social Login must be installed and active.
